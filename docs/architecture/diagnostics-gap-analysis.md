# Diagnostics And GAP Analysis

Diagnostics are required for both automation and paper playtesting.

## JSON Snapshot Requirements

- run metadata
- mission or skirmish id
- design seed and gameplay seeds
- objective transitions
- encounter and wave rolls
- loot and resource outcomes
- pathfinding anomalies
- stuck-unit and boundary failures
- fog and minimap counters
- performance counters
- failure summaries

## GAP Review Levels

- Macro
  - pacing
  - economy curve
  - mission flow
  - readability
- Meso
  - encounter quality
  - objective transitions
  - pressure and map control
  - tool availability
- Micro
  - pathing
  - collision
  - targeting
  - UI friction
  - visual ambiguity
