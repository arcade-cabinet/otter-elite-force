# Skirmish Harness

Skirmish is both a player mode and a deterministic systems-test harness.

## Required Uses

- spawn validation
- passable versus impassable boundary checks
- pathfinding validation
- build placement rules
- gather loops
- wave and encounter verification
- fog and minimap validation
- AI sandbox checks

## Presets

- macro
- meso
- micro

Each preset should carry deterministic seeds and emit diagnostics.
